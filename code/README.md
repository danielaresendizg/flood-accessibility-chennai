# Code (minimal)

This project is primarily a GIS + Space Syntax workflow (DepthmapX + QGIS). The code in this repository is intentionally minimal:

- `code/qgis/`: QGIS project files + styles (for reference; paths may need relinking)
- `code/python/`: small helper scripts used for flood-model input preparation / checks

Suggested structure:

```
code/
├── qgis/          # QGIS project notes, styles, processing steps
└── python/        # Optional scripts (e.g., cleaning, joins, exports)
```
